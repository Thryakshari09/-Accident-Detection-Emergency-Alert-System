# 🚑 Accident Detection & Emergency Alert System

## 📌 Description

This project is a simple simulation of an accident detection system built using Python. It detects sudden impact-like conditions using randomly generated sensor values and triggers an alert when a critical threshold is exceeded. The system then displays an emergency message along with location details, simulating a real-world accident alert system.

---

## 🧠 Technologies Used

* Python
* Random Module (for simulation)
* Time Module (for delay handling)

---

## ⚙️ How It Works

1. The system continuously generates random values to simulate sensor data (like vibration or impact).
2. Each value is printed on the screen as a "Sensor Value".
3. If the value exceeds a threshold (greater than 90), it is treated as an accident.
4. Once an accident is detected:

   * A warning message is displayed
   * Location details are shown
   * An alert message is triggered
5. The system stops after detecting the accident.
