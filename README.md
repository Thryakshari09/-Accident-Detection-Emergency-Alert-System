# 🚑 Accident Detection & Emergency Alert System

## 📌 Description

This project simulates an accident detection system using Python. It detects sudden impact and sends an alert message with location details.

## 🧠 Technologies Used

* Python
* Simulation

## ⚙️ How It Works

* Random values simulate sensor data
* High values indicate an accident
* Alert message is displayed

## ▶️ Run the Project

python accident_detection.py
